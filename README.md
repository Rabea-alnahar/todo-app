# Todo App (Vue + Node/Express + TypeScript)

A simple full-stack Todo application.

## Tech Stack
- Frontend: Vue 3 + Vite + TypeScript
- Backend: Node.js + Express + TypeScript
- Communication: REST API (JSON)

## Requirements
- Node.js + npm

## How to run (development)

### 1) Backend
```bash
cd backend
npm install
npm run dev
```
### 2) Frontend
```bash
cd frontend
npm install
npm run dev
```

### Run both (one command)
```bash
npm install
npm run dev
```
